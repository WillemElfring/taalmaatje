# Taalmaatje

How to start working on Taalmaatje.

1. Clone the repo.

2. CD into the "taalmaatje" folder.

3. Download composer packages.
```
composer install
```

4. Launch Laravel.
```
php artisan serve
```

5. Ready to use.
